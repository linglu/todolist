IDE 有用的快捷键：

一、F11 && Shift + F11

1. 使用 F11 标记行号
2. 使用 Shift + F11 在标记的行号间导航和切换

用处：
  在开发某个功能时，标记需要改动的若干处代码块，然后在标记的代码块间切换；

二、Ctrl + Alt + M 代码重构

1. 选中一块常用的、需要独立出来成为一个方法的代码；
2. 使用 Ctrl + Alt + M 进行代码重构，就选中的代码产生新的函数；
